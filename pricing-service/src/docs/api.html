POST /quotes/calculate

Calculate insurance quote with risk assessment and pricing. Performs BMI calculation, age validation, and underwriting risk classification.

Request Body
Field Type Required Description
productType string Required "term_life" (whole_life, annuity coming soon)
applicant.gender string Required "Male" or "Female"
applicant.birthDate string Required Format: YYYY-MM-DD
applicant.height number Required Height in cm (120-250)
applicant.weight number Required Weight in kg (35-200)
applicant.city string Required City of residence
applicant.usesNicotine boolean Required Nicotine use status
policy.termLength number Required 10 or 20 years
policy.coverageAmount number Required $250,000 - $1,500,000
Request Example
{
  "productType": "term_life",
  "applicant": {
    "gender": "Male",

    "usesNicotine": false

POST /contributions/validate

Validate contribution amounts against minimum requirements and calculate equivalents.

Request Body
Field Type Required Description
amount number Required Contribution amount in MAD
frequency string Required "monthly", "quarterly", "bi-annual", or "annual"
Minimum Contribution Requirements
  • Monthly: 250 MAD
  • Quarterly: 750 MAD (3 months)
  • Bi-annual: 1,500 MAD (6 months)
  • Annual: 3,000 MAD (12 months)
Request Example
{
  "amount": 500,
  "frequency": "monthly"
}
Response (200 OK - Valid)
{
  "validation": {
    "isValid": true,
    "monthlyEquivalent": 500,
    "annualTotal": 6000
  },
  "timestamp": "2025-01-15T10:30:45.123Z"
}
Response (200 OK - Invalid)
{
  "validation": {
    "isValid": false,
    "errorMessage": "Monthly contribution must be at least 250 MAD",
    "monthlyEquivalent": 100,
    "annualTotal": 1200
  },
  "timestamp": "2025-01-15T10:30:45.123Z"
}
Error Response (400 Bad Request)
{
  "error": "Invalid frequency",
  "message": "Frequency must be one of: monthly, quarterly, bi-annual, annual"
}
