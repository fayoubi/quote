POST /auth/request-otp

Request an OTP code for phone authentication. The code is valid for 5 minutes.

Request Body
{
  "phone_number": "612345678",  // Can also accept email
  "country_code": "+212"  // Optional, defaults to +212
}
Response (200 OK)
{
  "success": true,
  "message": "OTP sent successfully",
  "data": {
    "phone_number": "612345678",
    "otp": {
      "code": "123456",  // Only in development mode
      "expires_at": "2025-09-30T10:10:00Z"
    }
  }
}
Error Response (429 Too Many Requests)
{
  "status": "error",
  "message": "Too many failed attempts. Please try again later.",
  "lockedUntil": "2025-09-30T10:30:00Z"
}
