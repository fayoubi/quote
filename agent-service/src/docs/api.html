POST /auth/register

Register a new agent with their details. Phone number must be from Morocco (+212) or France (+33).

Request Headers
Content-Type: application/json
Request Body
{
  "firstName": "Ahmed",
  "lastName": "Bennani",
  "email": "ahmed.bennani@example.com",
  "countryCode": "+212",
  "phoneNumber": "612345678",
  "licenseNumber": "AG123456"  // Optional - auto-generated if not provided
}
Response (201 Created)
{
  "status": "success",
  "data": {
    "agent": {
      "id": "a1111111-1111-1111-1111-111111111111",
      "firstName": "Ahmed",
      "lastName": "Bennani",
      "email": "ahmed.bennani@example.com",
      "phoneNumber": "+212612345678",
      "countryCode": "+212",
      "licenseNumber": "AG123456",
      "isActive": true,
      "createdAt": "2025-09-30T10:00:00Z"
    }
  }
}
POST /auth/request-otp

Request an OTP code for phone authentication. The code is valid for 5 minutes.

Request Body
{
  "countryCode": "+212",
  "phoneNumber": "612345678",
  "deliveryMethod": "sms"  // Optional, defaults to 'sms'
}
Response (200 OK)
{
  "status": "success",
  "data": {
    "otpId": "otp-uuid-here",
    "expiresAt": "2025-09-30T10:05:00Z",
    "deliveryMethod": "sms",
    "code": "123456"  // Only in development mode
  },
  "message": "OTP sent successfully"
}
Error Response (429 Too Many Requests)
{
  "status": "error",
  "message": "Too many failed attempts. Please try again later.",
  "lockedUntil": "2025-09-30T10:30:00Z"
}
POST /auth/verify-otp

Verify the OTP code and receive JWT tokens for authenticated access.

Request Body
{
  "otpId": "otp-uuid-here",
  "code": "123456"
}
Response (200 OK)
{
  "status": "success",
  "data": {
    "agent": {
      "id": "a1111111-1111-1111-1111-111111111111",
      "firstName": "Ahmed",
      "lastName": "Bennani",
      "email": "ahmed.bennani@example.com",
      "phoneNumber": "+212612345678"
    },
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "refreshToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "expiresAt": "2025-10-01T10:00:00Z"
  },
  "message": "Authentication successful"
}
Error Response (401 Unauthorized)
{
  "status": "error",
  "message": "Invalid or expired OTP code"
}

POST /auth/logout üîí Requires Auth

Validate a JWT token (public endpoint for inter-service communication).

  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}
Response (200 OK)
{
  "status": "success",
  "data": {
    "valid": true,
    "agent": {
      "id": "a1111111-1111-1111-1111-111111111111",
      "phoneNumber": "+212612345678",
      "email": "ahmed.bennani@example.com",
      "firstName": "Ahmed",
      "lastName": "Bennani"
    }
  }
}
Error Response (401 Unauthorized)
{
  "status": "error",
  "data": {
    "valid": false,
    "reason": "Token expired"
  }
}
