### Database Access
```bash
# Pricing DB
psql -h localhost -p 5432 -U postgres -d pricing

# Enrollment DB
psql -h localhost -p 5433 -U postgres -d enrollment

# Agent DB
psql -h localhost -p 5434 -U postgres -d agent
```
